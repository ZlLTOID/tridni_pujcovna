Co je to programování?

Psaní návodu pro počítač

Programování = když člověk napíše přesný návod, co má počítač udělat — krok za krokem.

📝 Návod

Počítač nedělá nic „sám od sebe" — vždy jen to, co mu řekneš.

Jako recept na bábovku: nejdřív smíchej, pak peč — v přesném pořadí 🧁

🤖 Program

Návod uložený v souboru — třeba ve webové aplikaci nebo ve hře.

Jako návod na LEGO — z kostiček (kódu) postavíš hotovou věc 🧱

💻 Počítač

Čte návod a plní ho velmi rychle a přesně — milionkrát za sebou.

Jako robot, který nikdy nezapomene krok z návodu 🤖

Programování potkáš všude — ve hrách, na webu, v mobilech i v robotech. Dnes se naučíme, jak takový návod vypadá.

Proměnné — pojmenované krabičky

Kde program ukládá informace, se kterými pracuje

Proměnná = pojmenované místo v paměti. Dáš jí jméno a uložíš do ní hodnotu — později ji můžeš znovu přečíst nebo změnit.

📦 Přirovnání

Krabička s nápisem jmeno — uvnitř je „Honza". Můžeš obsah vyměnit za „Elišku".

$jmeno = "Honza";
$jmeno = "Eliška";  // změna

🎮 Příklady z programů

  • $jmeno — jméno hráče „Honza"
  • $body — skóre ve hře, třeba 150
  • $sviti — svítí lampička? ano/ne

Znak $ v PHP znamená „toto je proměnná". Každá proměnná má svůj datový typ — o tom hned další slide.

Datové typy — co ukládáme?

Jak různé druhy informací v paměti počítače

🔤 string

Text — slova a věty

Jako jméno na jmenovce ve škole 🏷️

"Honza", "Praha"

🔢 int

Celé číslo bez desetinné tečky

Jako body ve hře nebo počet žáků ve třídě 🎯

1, 28, 2026

✅ bool

Pouze ano nebo ne

Jako vypínač — svítí, nebo nesvítí 💡

true / false

📅 datetime

Datum a čas dohromady

Jako zápis v deníčku: „22. 6. v 10:00" 📓

2026-06-22 10:00

📋 pole (array)

Seznam více hodnot pod sebou

Jako seznam žáků na tabuli — jeden, dva, tři… 🧑‍🎓

["Honza", "Eliška", "Max"]

Příklad: jméno kamaráda (string), body ve hře (int), svítí světlo? (bool), kdy začíná přestávka (datetime), seznam spolužáků (pole)

Funkce — pojmenovaný kus návodu

Napíšeš jednou, použiješ mnohokrát

Funkce = skupina příkazů s jménem. Zavoláš ji a ona udělá svou práci — nemusíš psát stejný kód pořád dokola.

🔧 Příklad 1 — text

function pozdrav($jmeno) {
    return "Ahoj, " . $jmeno . "!";
}

echo pozdrav("Honzo");  // Ahoj, Honzo!
echo pozdrav("Eliško"); // Ahoj, Eliško!

$jmeno je parametr — hodnota, kterou funkci předáš.

🔧 Příklad 2 — dvě hodnoty

function celeJmeno($jmeno, $prijmeni) {
    return $jmeno . " " . $prijmeni;
}

echo celeJmeno("Honza", "Novák");
// Honza Novák

echo celeJmeno("Eliška", "Malá");
// Eliška Malá

Stejná funkce, jiné jméno — nemusíš psát skládání jmen znovu a znovu.

💡 K čemu to je

  • secti(2, 3) — vrátí 5, nemusíš počítat v hlavě
  • jeTeplo(28) — vrátí ano, když je nad 25 °C
  • prehrajZvuk() — udělá akci, třeba pípnutí ve hře

Funkce může něco vrátit (výsledek) nebo něco udělat (akce).

Ovládací struktury

Jak program rozhoduje a opakuje věci

if — když… tak…

Přirovnání: Když prší ☔ → vezmi deštník. Když neprší → jdi bez něj. 
if ($prsi) {
    vezmiDestnik();
} else {
    jdiVen();
}

Program si vybere jednu z možností.

foreach — pro každý…

Přirovnání: Paní učitelka volá třídu — každému žákovi řekne „Ahoj!" jeden po druhém 👋 
foreach ($ovoce as $kus) {
    echo $kus;
}

Projde celý seznam, položku po položce.

Příklad: Ve hře — pokud máš málo životů → ukaž varování. V cyklu projdi všechny kamarády a každému pošli pozdrav.

Co je to algoritmus?

Přesný postup krok za krokem

Algoritmus = seznam kroků, které vedou k cíli. Každý krok je jasný a dává smysl.

🌅 Příklad ze života

  1. Vstát z postele
  2. Umýt si zuby
  3. Obléct se
  4. Jít do školy

Když změníš pořadí (oblékneš se před vstáním?), přestane to fungovat.

🎮 Příklad v programu

  1. Hráč stiskne tlačítko „Skoč"
  2. if je na zemi → skoč, jinak nic nedělej
  3. Přičti body za skok
  4. Překresli postavu na obrazovce
  5. Přehraj zvuk „hop"

Algoritmus používá if a foreach — právě proto jsme se je učili!

Tři části spolupracují — každá má svou roli

Model Data skóre, jména, objednávky…
↩ Controller pošle výsledek zpět do View → uživatel vidí odpověď na obrazovce
  • View neřeší logiku — jen ukazuje a předává kliknutí
  • Controller propojuje obrazovku s daty
  • Model drží informace trvale (často v databázi)

🎯 Příklad: objednáš pizzu v aplikaci

View Menu s pizzami — vybereš „Šunková" a klikneš Objednat

Controller Zkontroluje, že máš vybranou pizzu, a pošle objednávku dál

Model Objednávka se uloží — jaká pizza, pro koho, v kolik hodin

View Obrazovka „Děkujeme, pizza je na cestě!" 🍕

CRUD — čtyři akce s daty

C — Create

Přidej nový kontakt do telefonu

R — Read

Zobraz seznam fotek nebo zpráv

U — Update

Uprav jméno kontaktu nebo skóre ve hře

D — Delete

Smaž starou fotku nebo zprávu

INSERT INTO kontakt (jmeno, telefon) VALUES (?, ?)  -- Create
SELECT * FROM fotka ORDER BY datum                    -- Read
UPDATE hra SET body = 100 WHERE hrac = ?              -- Update
DELETE FROM zprava WHERE id = ?                       -- Delete

Eventy — události

Na webu reagujeme na kliknutí, odeslání formuláře, najetí myší

Příklad: Klikneš na tlačítko ve hře → postava skočí. Odešleš zprávu kamarádovi → program ji uloží a ukáže na obrazovce.

V PHP: $_SERVER['REQUEST_METHOD'] === 'POST' = event „formulář odeslán"

SQL — jazyk pro databázi

Vytvoření tabulky

CREATE TABLE zak (
  id INTEGER PRIMARY KEY,
  jmeno TEXT NOT NULL,
  prijmeni TEXT NOT NULL,
  trida_id INTEGER
);

Čtení dat

SELECT * FROM zak
WHERE trida_id = ?
ORDER BY prijmeni;

SQL píšeme přímo v Controlleru — je přehledné a vidíme, co se děje.

Zkus si to doma ve Scratchi!

Programování z barevných kostiček — zdarma v prohlížeči

Scratch je program, kde skládáš návod pro počítač z puzzle kostiček — nemusíš psát složitý text jako v PHP.
  • Funguje v prohlížeči — stačí počítač nebo tablet s internetem
  • Úplně zdarma na scratch.mit.edu
  • Stejné myšlenky jako u nás: události, if, opakování

Jak začít se Scratchi doma

Postup krok za krokem

  1. 1Otevři scratch.mit.edu v prohlížeči (Chrome, Firefox, Edge…)
  2. 2Klikni na Připojit se ke Scratchi a založ si účet (s pomocí rodičů)
  3. 3Klikni Vytvoř → otevře se editor s kočkou uprostřed
  4. 4Vlevo vidíš barevné kostičky — přetahuj je doprostřed a skládej je k sobě
  5. 5Nahoře je zelená vlajka ▶ = spustit program, červené kolečko ⏹ = zastavit

Projekt se ukládá sám do tvého účtu — můžeš se k němu kdykoli vrátit.

Co kde ve Scratchi najdeš

🟡 Události

Kdy se program spustí

„Po kliknutí na zelenou vlajku" — jako kliknutí na webu

🔵 Pohyb

Kam se postava posune, otočí…

„Posuň se o 10 kroků" — jako krok v algoritmu

🟣 Vzhled

Barva, velikost, kostýmy

Změna vzhledu na obrazovce — jako View u nás

🟠 Řízení

Opakování, čekání, if / jinak

„Dokud… opakuj" a „když… tak" — přesně jako if a foreach!

Dole je scéna (obrazovka) a postavy (sprites) — každá má svůj vlastní skript.
